# MedVerify: Medical Hallucination Detector

This repository contains the code for the MedVerify project.

## Demo Option 1: Python App

**Recommended for reviewer demo**

**Prerequisites:** Python, Streamlit, spaCy, requests

1. Open a terminal in the project folder:
   `cd python_project`
2. Run the Streamlit app:
   `streamlit run app.py`

## Demo Option 2: Web App

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
